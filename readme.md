### installation

install `reminders-cli`

```
brew install keith/formulae/reminders-cli
```

add alias (optional, replace with your path and command)

```
echo "alias remind=\"python ~/documents/projects/github/remind/main.py/\"" >> ${ZDOTDIR:-$HOME}/.zshrc
```
